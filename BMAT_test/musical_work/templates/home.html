Welcome to MusicalWork!


Part 1:

The matching and reconciliation script can be found under /musical_work/management/commands/import_csv_data.py

This management command requires as parameter a link to a remote CSV file.

E.g. python manage.py import_csv_data http://localhost:8080/static/works_metadata.csv

If the Docker container is up and running, this command can be invoked from your host with the following syntax:

docker-compose run backend python3 manage.py import_csv_data http://localhost:8080/static/works_metadata.csv


Part 2

See the list of Musical Works here

You can also access a single Musical Work metadata at http://localhost:8080/v1/musicalwork/{ISWC}

Import a CSV file: send a POST request to http://localhost:8080/v1/import_csv_report_view/ including in the body a valid CSV file formatted as for works_metadata.csv example

Want to retrieve metadata about some Musical Works? Send a GET request to http://localhost:8080/v1/export_csv_report_view/?iswc={VALUE}&iswc={ANOTHER_VALUE}&iswc=... to download a CSV file containing the metadata about all the requested Musical Works
